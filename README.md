# Sri Kamala Hospital - Professional Healthcare Platform

**Official website:** [https://srikamalahospital.online](https://srikamalahospital.online)

A modern, full-stack hospital management and public information website built with **React**, **Tailwind CSS**, and **Node.js**.

## Features

- **Professional UI/UX**: Premium hospital-themed design with smooth animations and high responsiveness.
- **OP Appointment Booking**: Integrated system for booking out-patient appointments.
- **Live Status**: Real-time hospital status and clock.
- **Specialized Doctors**: Professional profiles for medical staff.
- **Patient Reviews**: Testimonials and ratings.
- **Admin Dashboard**: (In development) Management of appointments and patient data.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Lucide React.
- **Backend**: Node.js, Express, MongoDB, Mongoose.
- **Real-time**: Custom hooks for live time and status.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   cd server && npm install
   cd ../client && npm install
   ```

2. **Setup Environment**:
   Create a `.env` file in the `server` directory with your MongoDB URI and other credentials (see `.env.example`).

3. **Run Development Server**:
   ```bash
   npm run dev
   ```
